This page contains versions of the Bioschemas types that are ready for integration into Schema.org.

Other types on which the Bioschemas community are still discussing are published via the drafts page.

Should you wish to contact the community to discuss our efforts please email public-bioschemas@w3.org.

Types

Schema.org describes ‘types’ of information, which then have ‘properties’. The types are things that we can talk about and the properties are the things that we can say about the type. We have found it necessary to propose some additional types to the Schema.org vocabulary to describe life sciences resources.

The types specified below in the Release tab are Release Candidates (RC) for inclusion into Schema.org. They are currently undergoing final review. Comments on the mailing list or GitHub issue tracker are encouraged.
