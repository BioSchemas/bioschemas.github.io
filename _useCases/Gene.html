Use Cases for {{page.name}}

{% assign group = site.groups | where:"identifier", page.group %} {% for g in group %}

The following use cases were developed by the {{ g.name }} group. {% endfor %}

This use cases apply to the following biological types: “gene”.

Findability

As a user, I would like to find relevant/well-known resources regarding genes when looking by gene name or gene identifier.

Summarization

As a user, whenever I look for a gene, I would like to see a quick summary similar to the one provided for GeneWiki pages but including more resources.

Link to external resources

As a user, I want to be able to navigate from gene resources to other relevant and related resources.
