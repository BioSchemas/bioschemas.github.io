accessibilityFeature

Canonical URL: http://schema.org/accessibilityFeature

Thing > Property > accessibilityFeature

Content features of the resource, such as accessible media, alternatives and supported enhancements for accessibility (WebSchemas wiki lists possible values).

Usage: Between 100 and 1000 domains
  • Leave public feedback on this term 💬
  • Check for open issues.
  • 'accessibilityFeature' is mentioned in layer: core
Values expected to be one of these types
Text
Used on these types
CreativeWork
