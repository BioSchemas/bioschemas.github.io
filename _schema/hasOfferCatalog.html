hasOfferCatalog

Canonical URL: http://schema.org/hasOfferCatalog

Thing > Property > hasOfferCatalog

Indicates an OfferCatalog listing for this Organization, Person, or Service.
  • Leave public feedback on this term 💬
  • Check for open issues.
  • 'hasOfferCatalog' is mentioned in layer: core
Values expected to be one of these types
OfferCatalog
Used on these types
Organization
Person
Service
