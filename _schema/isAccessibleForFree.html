isAccessibleForFree

Canonical URL: http://schema.org/isAccessibleForFree

Thing > Property > isAccessibleForFree

A flag to signal that the item, event, or place is accessible for free.
  • Leave public feedback on this term 💬
  • Check for open issues.
  • 'isAccessibleForFree' is mentioned in layer: core
Values expected to be one of these types
Boolean
Used on these types
CreativeWork
Event
Place
PublicationEvent
Supersedes
free


Examples
